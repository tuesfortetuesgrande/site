# Tú és Forte, Tú és Grande

Rede social para torcedores do São Paulo FC.

## 🚀 Tecnologias
- HTML5, CSS3, JavaScript
- Firebase (Auth, Firestore, Storage)
- Cloudflare Workers

## 📦 Instalação
1. Clone o repositório
2. Abra o `index.html` no navegador
3. Ou faça deploy com `npx wrangler deploy`
